Web Services

The data for this fieldwork project is accessible via the internet. The easiest access is via Web Map Services (WMS) and Web Feature Services (WFS). These services let you load online geodata in the GIS client (such as QGIS or ArcGIS) on your desktop. The url for map services (both WMS and WFS) for your project data is:

WMS and WFS location: {{request.scheme}}://{{request.host}}/wms/{{cu.wms_key}}

In QGIS you can add your data to a QGIS project by using the Layer Add Layer Add WFS Layer menu and entering the above URL. It is also possible to add the layer a WMS service (which shows only the map image, not feature data).
