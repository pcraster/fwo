Map services

The data for this fieldwork project is available via the internet. The easiest access is via Web Map Services (WMS) and Web Feature Services (WFS). These services let you load online geodata in the GIS client (such as QGIS or ArcGIS) on your desktop. The url for map services (both WMS and WFS) for your project data (as well as the base maps) is:

http://{{request.host}}/projects/{{project.slug}}/{{user.id}}/mapservices

In QGIS you can add your data to a QGIS project by using the Layer Add Layer Add WMS/WFS Layer menu and entering the above URL.

Sharing your web map

If you would like to share only your web map with someone else, like an external supervisor, you can create a shortened URL through which the map can be accessed. Please be aware that short URLs allow users to view your web map without having to enter a password!
